# Programación en Ensamblador 8086

Este repositorio se ha creado para documentar el proceso de aprendizaje sobre el uso del lenguaje de programación ensamblador, específicamente para el microprocesador 8086.

## Sobre el Lenguaje Ensamblador

El ensamblador es un lenguaje de programación de bajo nivel utilizado para escribir programas que están muy cerca del lenguaje de máquina. A diferencia de los lenguajes de alto nivel, el ensamblador permite un control preciso sobre el hardware del ordenador, lo que lo hace ideal para tareas que requieren una optimización extrema del rendimiento. En este repositorio, aprenderemos sobre la sintaxis del ensamblador, cómo escribir y depurar programas en este lenguaje, y su aplicación en la programación de sistemas.

## Sobre el Microprocesador Intel 8086

El Intel 8086 es un microprocesador de 16 bits introducido por Intel en 1978. Fue la base para la arquitectura x86 y marcó un avance significativo en la tecnología informática. Aquí exploraremos su arquitectura, conjunto de instrucciones y su importancia histórica en el desarrollo de los sistemas informáticos modernos.
